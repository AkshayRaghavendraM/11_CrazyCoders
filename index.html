Frequently Asked Questions

What devices are compatible with Hologram Arena? +
Hologram Arena works on any device with a modern web browser that supports WebGL 2.0. This includes desktop computers, laptops, tablets, and smartphones. For the best experience, we recommend using Chrome, Firefox, Safari, or Edge browsers.
Do I need special hardware for holographic viewing? +
No special hardware is required! Hologram Arena creates the illusion of 3D holographic projections using advanced rendering techniques on your regular screen. However, for enhanced experiences, you can use VR headsets or 3D displays.
Is there a free version available? +
Yes! Hologram Arena offers a free tier with basic holographic creation tools and limited storage. Premium features include advanced rendering options, unlimited storage, and priority support.
Can I integrate Hologram Arena into my own application? +
We provide a comprehensive API and SDK that allows you to integrate holographic capabilities into your own applications. Documentation and code examples are available in our developer portal.
What file formats are supported for import/export? +
Hologram Arena supports popular 3D formats including OBJ, FBX, GLTF, and STL for import. You can export your creations as interactive web components, video files, or 3D model formats for use in other applications.
